## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Node.js
- React

### Setup
```
git clone https://github.com/petudeveloper/user-skills.git
cd user-skills
```
### Install
```
npm install
```

### Usage
```
npm start
```
